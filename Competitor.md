## 2. Navigation & Category Structures

### Pattern A: Gender-First Navigation (Most Common)

Used by: **Borjan, Metro Shoes, Flash Footwear, Jutay, Insignia (partially)**

```
Home | MEN | WOMEN | KIDS | Sale
```

- Primary nav splits by **gender/audience first**
- Within each gender, subcategories appear (Sneakers, Formal, Casual, Sandals, etc.)
- Borjan adds "NAYZA" (sub-brand) and "Shop by Collection" as separate nav items
- Insignia has: WOMEN | MEN | BAGS | FRAGRANCES as top-level tabs

**Borjan nav structure:**
```
Home | Category (mega menu) | Cart | Search
Category dropdown:
  → MEN (Sneakers, Formal, Casual, Sandals, Slippers, Chappal, Moccasin, Peshawari)
  → WOMEN (Sneakers, Heels, Sandals, Slippers, Chappal, Court Shoes, Moccasin, Pumps)
  → KIDS
  → BAGS (Handbags, Clutches)
  → ACCESSORIES
```

**Insignia nav structure:**
```
WOMEN | MEN | BAGS | FRAGRANCES
Homepage sections: New Arrivals, The Drop, Trending Now, Capsule Edit
```

### Pattern B: Product-Type-First Navigation

Used by: **1st Step, Unze London, ECS, Walkeaze**

```
Home | FOOTWEAR | BAGS | APPAREL | ACCESSORIES | FRAGRANCES
```

- Primary nav splits by **product type first**, then gender within
- Better suited for multi-category retailers (shoes + bags + clothing)

**1st Step nav structure:**
```
Men's Footwear | Women's Footwear | Apparel | Boy's Footwear | Girl's Footwear | Clutches | Hand Bags
Homepage sections: Customer's Favourite, Men's Best Sellers, Women's Best Sellers, Apparel, Clutch, Handbags, Boys, Girls
Footer categories: Sneakers, Pret, Pumps, Sandals, Slippers, Bags, Court Shoes
```

**Unze London nav structure:**
```
Women Shoes | Women Clothing | Women Bags | Girl Clothing | Men Shoes | Men Clothing | Boy Clothing
Within Women Shoes: Sandals, Courts, Block Heels, Heel Sandals, Comfort Sandals, Slippers, Khussa, Comfort Slippers, Moccasins, Ballerinas, Boots, Loafers, Trainers
Within Women Bags: Handbags, Bridal/Party Bags, Casual Bags, Shoulder Bags, Cross Body, Top Handle
Within Men Shoes: Formals, Moccasin, Slippers, Sandals, Peshawari, Boots, Trainers
```

**ECS nav structure:**
```
Women Footwear | Kids | Bags & Clutches | Fragrances | Accessories | Apparel
Within Women Footwear: Sandals, Slippers, Washables, Khussa, Plus Sizes, Pumps, Loafers, Court Shoes, Sneakers, Back Open, Boots, Bridals, Heels, Wedges
```

### Pattern C: Hybrid / Mega-Menu with Both Axes

Used by: **Stylo, NDURE, Clive**

**Stylo nav structure:**
```
Women | Men | Kids | Bags | Sale
Homepage sections: Chamak Dhamak (women shoes), Pumps, Moccasin, Court Shoes, Heels, Sneakers, Kids Collection, Bags Collection, Fragrances
Mega-menu under Women: Shoes by type (Sandals, Slippers, Pumps, Sneakers, etc.) + Apparel + Accessories
Mega-menu under Men: Shoes by type + Clothing
```

**NDURE nav structure:**
```
NEW IN | SLIDES | SANDALS | SNEAKERS | LOAFERS & MOCCS | ALL SHOES
Then within each: Men / Women / Kids filters
Collections page has 100+ named collections (by season, gender, type, sale, price range)
```

**Clive nav structure:**
```
Flat 50% Off | Ladies | Kids | Men | Bags | Flat 70% Pret | Winter Collection
Within Ladies: Comfort Wear, Chappals, Moccasin/Loafers, Sneakers, Back Open, Peep Toes, Day To Day Wear, Formal Wear, Pacer, Ethnic Wear, Signature, Festive Collection

### Pattern D: Occasion-Based Navigation

Used by: **Milli Shoes**

```
MEN → Men's Occasion (Casual, Party, Sports, Wedding, Formal) | Men Shoe Type (Boots, Slippers, Peshwari, Sneakers, Leather Shoes, Imported Shoes, Moccasins)

## 4. Product Listing Page (PLP) Patterns

### Filter Sidebar (Most Common)

Used by: **Stylo, NDURE, Clive, Hopkicks, ECS**

Typical filters:
- **Size** (EU sizes for shoes: 39-46 men, 36-41 women, 20-35 kids)
- **Color** (with color swatches)
- **Price** (range slider or brackets)
- **Availability** (In stock / Out of stock)
- **Product Type** (Sneakers, Formal, Casual, Sandals, etc.)
- **Brand** (for multi-brand stores)

### Sort Options (Universal)

Featured, Best Selling, Price Low→High, Price High→Low, Alphabetically A→Z, Date New→Old

### Product Card Patterns

| Element | Prevalence |
|---------|------------|
| Product image | 100% |
| Product name | 100% |
| Price (PKR) | 100% |
| Original price (strikethrough) + discount % | ~80% (when on sale) |
| Quick view / Quick shop button on hover | ~60% |

## 5. Product Detail Page (PDP) Patterns

### Size Selection

- **Shoes**: EU sizes displayed as clickable chips/buttons (39, 40, 41, 42, 43, 44, 45, 46)
- **Women's shoes**: EU 36-41 (sometimes 42)
- **Kids**: EU 20-35 or age-based
- **Bags**: S/M/L/XL or One Size
- Out-of-stock sizes shown as crossed out or disabled
- Some sites (Borjan, NDURE) show size availability directly on product card

### Key PDP Elements

1. Image gallery (main + thumbnails, sometimes carousel)
2. Product name + price (with original price strikethrough if on sale)
3. Size selector (chip buttons)
4. Color selector (if multiple colors)
5. Add to Cart button
6. BNPL installment display ("Pay only Rs. X now" — BaadMay integration)
7. Product description
8. Delivery information (3-5 working days)
9. Exchange/return policy snippet
10. Related products carousel

### BNPL (Buy Now Pay Later)

**BaadMay** integration is extremely common across Pakistani shoe e-commerce:
- Borjan, Stylo, Clive, Walkeaze, Unze all show "Pay in 3 Easy Installments"
- Displayed as: "Pay only Rs. X now" alongside the full price
- Checkout option: "Select BaadMay at Checkout"

## 7. Payment & Checkout Patterns

### Payment Methods (Universal across Pakistan)

1. **Cash on Delivery (COD)** — Available on virtually every site
2. **Credit/Debit Card** — Via gateway (JazzCash, EasyPaisa, bank transfer)
3. **BaadMay (BNPL)** — 3-installment buy-now-pay-later, shown prominently
4. **JazzCash / EasyPaisa** — Mobile wallet payments
5. **Bank Transfer** — Direct deposit

### Checkout Flow

Typical: Cart → Checkout (Shipping Info + Payment) → Order Confirmation  
Some sites (Stylo, ECS) require account creation; others allow guest checkout.

### Shipping

- **Free shipping threshold**: Rs. 2,500–5,000 (varies by brand)
- **Standard delivery**: 3-7 working days
- **Express shipping**: Available on some (TNV, Hopkicks)

### Return/Exchange Policy

- **7-day exchange** is standard (Borjan, 1st Step, Flash Footwear, NDURE)
- Some offer replacement within 48 hours (Datchi)
- Most do NOT offer refunds — exchange only

## 9. Category Hierarchy Comparison

### How competitors organize beyond shoes

```
SHOES-ONLY BRANDS (traditional):
  Gender → Shoe Type
  e.g., Men → Sneakers / Formal / Casual / Sandals / Slippers

MULTI-CATEGORY BRANDS (emerging):
  Product Type → Gender → Subcategory
  e.g., Shoes → Men → Sneakers
       Bags → Women → Handbags
       Accessories → Unisex → Belts

INTIKHAB'S CURRENT APPROACH:
  Product Type → Category (Gender) → Subcategory
  e.g., /shoes/men/sneakers
       /bags/women/handbags
       /accessories/unisex/wallets
```

**This matches the emerging multi-category pattern** used by 1st Step, Unze, ECS, and Walkeaze — which is the right direction as Intikhab expands beyond shoes.

## 11. Competitive Gaps & Opportunities for Intikhab

### What competitors lack (Intikhab can capitalize on)

1. **Hierarchical URL structure** — No competitor uses clean `/shoes/men/sneakers` URLs. Most use `/collections/men-sneakers` (Shopify default). Intikhab's SEO-friendly URLs are a differentiator.
2. **Per-size stock visibility** — Most competitors show sizes as available/unavailable but don't show actual stock counts. Intikhab's `sizeStock` system can show "Only 2 left!" for urgency.
3. **Admin product management** — Most competitors manage products through Shopify admin. Intikhab's custom admin dashboard with cascading category selection and shoe-specific size grid is more tailored.
4. **Modern SPA-like experience** — Shopify sites have full page reloads. Next.js App Router provides faster navigation with client-side transitions.
5. **Product type awareness** — No competitor clearly segments shoes/bags/accessories/clothing in their URL or navigation. Intikhab's `productType` field enables this.

### What Intikhab should match (table stakes)

1. **BaadMay BNPL integration** — Shoppers expect "Pay in 3 installments" display
2. **Size chips on product cards** — Standard across top brands
3. **Quick view / Quick shop** — Expected UX pattern
4. **Shop by Price sections** — "Under Rs. 2000", "Under Rs. 3000"
5. **7-day exchange policy** — Industry standard
6. **Free shipping threshold** — Rs. 3,000-5,000 range
7. **Announcement bar** — For sales, new arrivals, shipping info
8. **Instagram feed section** — Social proof on homepage
9. **Size chart on PDP** — Reduces return rate
10. **Occasion-based browsing** — Wedding, Formal, Casual, Party filters

## 12. Summary: Recommended Navigation for Intikhab

Based on competitor analysis, Intikhab's enhanced category structure aligns with the **multi-category pattern** (Product Type → Gender → Subcategory). The recommended top-level navigation:

```
SHOES | BAGS | ACCESSORIES | CLOTHING | SALE
```

With mega-menu dropdowns:
```
SHOES → Men (Sneakers, Formal, Casual, Sandals, Sports, Boots)
      → Women (Sneakers, Heels, Sandals, Slippers, Court Shoes, Khussa)
      → Kids (Boys, Girls)
      
BAGS → Women (Handbags, Clutches, Shoulder Bags, Backpacks)
     → Men (Briefcases, Backpacks)

ACCESSORIES → Belts, Wallets, Socks, Laces, Insoles

CLOTHING → Men (T-Shirts, Shirts, Pants, Jackets)
         → Women (Tops, Dresses, Pants)
```

URL structure: `/{productType}/{category}?subcategory={subcategory}`  
This is cleaner and more SEO-friendly than any competitor currently offers.
